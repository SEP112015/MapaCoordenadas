# Mapa con Coordenadas

Aplicación móvil desarrollada en Flutter que permite:

- Ingresar nombre, apellido, latitud y longitud
- Mostrar un mapa con un marcador
- Mostrar información del usuario y ubicación al tocar el marcador

## Tecnologías usadas
- Flutter
- Google Maps API
- Geocoding
